# Certezza Infinance - Sitio Web

Sitio web desarrollado para **Certezza Infinance**, una franquicia de brokers financieros en México.  
El objetivo del proyecto es mejorar la presencia digital de la franquicia y facilitar la captación de **clientes potenciales** y **nuevos asesores financieros** mediante un sistema de contacto y agendamiento de citas.

---

## 🚀 Descripción del Proyecto

Este sitio web fue desarrollado como una solución digital para ayudar a la franquicia **Certezza Infinance** a promocionar sus servicios financieros y optimizar su proceso de generación de prospectos.

El sitio permite a los visitantes:

- Conocer los servicios financieros ofrecidos
- Solicitar información
- Agendar citas
- Postularse para unirse al equipo de asesores

La página fue diseñada para ser **simple, clara y optimizada para conversión**, enfocándose en que los visitantes puedan contactar rápidamente al equipo de la empresa.

---

## 🎯 Objetivos del Proyecto

- Mejorar la presencia digital de la franquicia
- Facilitar el contacto con clientes potenciales
- Atraer nuevos asesores financieros
- Presentar los servicios de forma clara y profesional
- Centralizar las solicitudes de contacto y citas
